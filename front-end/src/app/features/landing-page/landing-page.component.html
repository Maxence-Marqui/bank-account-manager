Bienvenue sur le projet
Bank Account Manager

Cette application développée en Java Spring Boot avec Hibernate et JPA, utilisant une base de données PostgreSQL et un front-end en Angular, a pour objectif d'appliquer les bonnes pratiques de ces technologies pour créer une solution bancaire complète.

Pour les utilisateurs :

  • - Créer des utilisateurs
  • - Gérer leurs comptes
  • - Effectuer leurs transactions

Pour les administrateurs :

  • - Avoir une vue d'ensemble sur les activités
  • - Gérer les activités des clients
  • - Modifier les données utilisateurs
  • - Garder un œil sur les activités suspectes
